# homebrew-guideai

Homebrew tap for [GuideAI](https://github.com/SandRiseStudio/guideai) — AI-powered developer tooling and task orchestration CLI.

## Install

```bash
brew tap sandrisestudio/guideai
brew install guideai
```

## Update

```bash
brew update
brew upgrade guideai
```

## Uninstall

```bash
brew uninstall guideai
brew untap sandrisestudio/guideai
```

## About

This tap installs GuideAI from [PyPI](https://pypi.org/project/guideai/) into an isolated Homebrew-managed Python virtualenv.
